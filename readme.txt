Ele inclui os divs antes e depois do texto dos artigos e/ou dos comentários. A inclusão do código no rodapé do blog também é feita pelo plugin. A única configuração necessária a ser feita é informar o número do site de origem na página de administração que é disponibilizada ao se habilitar o plugin.Você pode escolher determinados artigos para que não sejam mostrados anúncios do Dynabox.O plugin inclui ainda créditos para o Dynabox e para mim, o autor do plugin. Acabei incluindo a nota de rodapé após ler o artigo onde fica claro que ninguém prestigia o autor de plugins. A inspiração foi o [artigo do Matt](http://photomatt.net/2007/04/12/plugin-authors-get-no-love/ "Authors Get No Love"). Junto com a publicação desta nota de rodapé é publicado também a parte mais importante do código do Dynabox. é o código onde está o código do site de origem.Se você; tiver problemas com o plugin ou quer sugerir alguma modificação, por favor acesse a página de [contato](http://celular.ellgrupo.com/blog/contato/) .== Installation ==A instalação do plugin é bastante simples.1. Faça upload da pasta **WP-Dynabox** para a pasta **/wp-content/plugins/**2. **Ative** o WP-Dynabox no menu **Plugins** do Wordpress3. **Informe o seu código do site de origem** e personalize a exibição dos anúncios**Importante!**Para que o plugin funcione corretamente, é **necessário** que o seu tema tenha uma chamada para função **wp_footer()**. Se você; não sabe como incluir a chamada, veja [aqui](http://www.wpdesigner.com/2007/05/30/where-exactly-should-you-place-wp_footer/ aqui). é necessário também que você; tenha uma conta ativa no [Dynabox](http://www.Dynabox.com.br/ "Dynabox").== Frequently Asked Questions ==   FAQ - Perguntas Frequentes = Não aparecem anúncios nos meus artigos. Verifiquei o código da página do artigo e os códigos do Dynabox não estão sendo incluídos. O que eu faço? = O problema pode estar no tema do seu blog, garanta que existe a chamada "wp_footer()" no arquivo de tema "Rodapé" (ou footer, caso seu wordpress esteja em inglê;s) do seu tema ativo. Veja mais [aqui](http://www.wpdesigner.com/2007/05/30/where-exactly-should-you-place-wp_footer/ aqui) = Preciso me cadastrar em algum lugar para usar este plugin? = Sim! é necessário ter uma conta ativa no progrma de afiliados do buscapé para usar a ferramenta [Dynabox](http://afiliados.buscape.com.br/afiliados/Dynabox.do "Dynabox") para que o plugin funcione como esperado. = Posso determinar que não sejam mostrados anúncios do Dynabox em alguns artigos? = Se você; quiser que algum artigo não receba anúncios do Dynabox, basta escolher a opção **Não mostrar anúncios do Dynabox neste artigo** na página de edição de artigos.== Screenshots ===== Changelog === 1.0.0 =* 2009-07-17* Primeira versão com funcionalidades básicas.
